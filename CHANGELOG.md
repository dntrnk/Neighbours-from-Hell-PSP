# Changelog

## [0.2.1] - 11-07-2026

### Добавлено
- Разная громкость для звуков.
- Разный приоритет для звуков: теперь звуки, которые весят много, не будут выгружаться во время игры.

### Изменено
- Отключено управление камерой через стик после завершения уровня.
- Улучшена производительность отрисовки за счёт добавления Lazy Batching.

### Исправлено
- Исправлена выгрузка звуков, которые дольше всех не использовались.
